# Purpose

This register records the governance information for all datasets used by Artificial Intelligence systems within {{COMPANY_NAME}}.

It provides objective evidence that data used by AI systems is identified, classified, reviewed and governed in accordance with Regulation (EU) 2024/1689 (EU AI Act) and Regulation (EU) 2016/679 (GDPR).

# Data Classification Levels

The following data classifications are used:

- Public
- Internal
- Confidential
- Restricted

---

# Data Quality Criteria

Each dataset shall be periodically evaluated for:

- Accuracy
- Completeness
- Consistency
- Integrity
- Timeliness
- Relevance

---

# Review Requirements

The register shall be updated whenever:

- a new AI system is introduced;
- a new data source is added;
- data classification changes;
- data quality issues are identified;
- regulatory requirements change.
